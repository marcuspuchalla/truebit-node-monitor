TrueBit Node Monitor

A decentralized, community-developed monitoring solution

Important Notice

Independent Project

This is a community-developed monitoring tool.

It is NOT officially affiliated with, endorsed by, or associated with TrueBit Foundation or any official TrueBit entities.

Use entirely at your own risk.

What is TrueBit Node Monitor?

  • Real-time monitoring dashboard for your TrueBit node
  • Track tasks, invoices, and node activity
  • Beautiful web interface with live updates
  • Optional federation network for global insights
  • Runs as a Docker container alongside your node

Key Features

Real-Time Monitoring

Live task tracking, log streaming, and instant status updates via WebSocket

Privacy First

Your wallet and private keys are never accessed, stored, or transmitted

Federation Network

Opt-in global network to see activity across all participating nodes

Docker Native

Single command deployment, runs alongside your TrueBit runner node

Quick Start

# Clone the repository
git clone https://github.com/marcuspuchalla/truebit-node-monitor.git

# Start the monitor
docker compose -f docker-compose.monitor.yml up -d

# Open your browser
open http://localhost:8090

What You'll See

Dashboard

Node status, recent tasks, invoice summaries at a glance

Tasks View
